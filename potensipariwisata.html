Paket Pariwisata

Pengelolaan bersama antara 3 (tiga) pilar dengan Pemerintah Desa Jagalan terhadap potensi yang ada diantaranya dengan memabuat Paket-paket Wisata Jagalan, diantaranya :

1. Paket Kirab Budaya

Paket Wisata pada Acara Kirab Budaya yang dilaksanakan setiap tahun pada bulan April.Beberapa kegiatan muali dari menyaksikan pertunjukan kesenian budaya Jagalan sampai dengan Prosesi Kegiatan “Nawu Sendang” (membersihkan sumber mata air). Fasilitas yang diberikan diantaranya : Tour Guide, menginap selama 3 hari, makan 3 kali per hari dengan harga paket perorang Rp. 1.200.000,- *)

2. Paket Bikin Perak

Paket Wisata ini para wisatawan akan dikenalkan dengna seni pembuatan handycraft dari logam perak. Para wisatawan akan mengetahui posesi pembuatan kerajinan ini, bahkan wisatawan akan belajar/praktek membuat kerajinan ini dan hasil karya wisatawan dapat dibawa pulang sebagai cinderamata. Fasilitas yang diberikkan diantaranya : Tour Guide, Snack, Cinderamata dengan harga paket per orang Rp. 200.000,- (Domestik) dan Rp. 250.000,- (Manca) *)

3. Paket Tlisih

Paket Wisata ini merupakan paket jelajah Desa Jagalan, para wisatawan akan dikenalkan dengan keramahan buadaya sopan-santun dan mengenal potensi Desa Jagalan lebih dekat. Wisatawan akan berinteraksi langsung dengan warga untuk mengetahui nilai serta obyek yang ada. Fasilitas untuk Paket ini diantaranya :Guide Tour, Snack, dan Peta Jagalan dengan harga paket per orang Rp. 85.000,- (domestic) dan Rp. 135.000,- (manca) *)

4. Paket Religi

Paket Wisata ini para wisatawan akan mengenal obyek wisata sekitar Masjid Besar Mataram dan mengenal perilaku, budaya dan sosial yang ada. Fasilitas untuk paket ini diantaranya : Tour Guide, Sewa Kostum, Snack, dan Merchandise dengan harga paket per orang Rp. 70.000,- (domestic) dan Rp. 100.000,- (Manca) *)

5. Paket Ngonthel

Paket Wisata ini merupakan paket wisata untuk mengenal Jagalan Tempo dulu, dimana para wisatawan kan diajak mengelilingi Desa Jagalan dengan menggunakan transportasi sepeda “onthel” untuk mengenal dengan mengunjungi semua bangunan tempo dulu. Fasilitas yang diberikan diantaranya : Tour Guide, Sewa Sepeda Jawa, dan Snack dengan harga paket per orang Rp. 85.000,- (domestic) dan Rp. 135.000,- (manca) *)

Selain paket-paket wisata, pengelolaan juga melakuka menggelar workshop-workshop untuk wisatawan ataupun pelaku usaha baru untuk mengetahui prosesi seni maupun pembuatan makanan tradisional khas Jagalan. Beberapa wosrkshop yang sudah dilaksanakan di Desa jagalan diantaranya :Kuliner, Mahar Uang, Kerajinan Logam Perak dsb.
